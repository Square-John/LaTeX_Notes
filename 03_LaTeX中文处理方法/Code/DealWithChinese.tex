	勾股定理：直角三角形的斜边长度的平方等于两腰的各自长度的平方之和。
	
	设三角形$ ABC $，其中$ \angle C = 90\degree $，则有
	\begin{equation}	
	AB^2 = 	BC^2 + AC^2.	
	\end{equation}
